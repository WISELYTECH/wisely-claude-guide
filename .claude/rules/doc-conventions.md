# 문서 컨벤션 규칙

이 저장소의 마크다운 문서를 수정할 때 반드시 다음 규칙을 따르세요.

## 필수 규칙

1. **한국어 작성** — 본문은 한국어. 명령어, 코드, 기술 용어 원문은 영어 유지
2. **코드블록 언어 태그** — 모든 코드블록에 언어 태그 필수 (```bash, ```json, ```yaml, ```markdown 등). 언어 태그 없는 코드블록 금지
3. **업데이트 날짜** — 문서 마지막 줄에 `**마지막 업데이트**: YYYY-MM-DD` 형식 필수. 문서를 수정하면 날짜도 오늘로 갱신
4. **문서 시작 구조** — `# 제목` 다음에 `> 한 줄 요약` 그리고 `---` 구분선
5. **비엔지니어 친화** — 기술 용어를 처음 사용할 때 간단한 설명 포함. 예: "MCP(Model Context Protocol) — 외부 도구를 연결하는 표준"
6. **섹션 구분** — 주요 섹션 사이에 `---` 구분선 사용
